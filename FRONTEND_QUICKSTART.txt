API CONNECTION
==============

Backend is already set up for CORS âœ…

Your frontend can call backend:
  
  import axios from 'axios';
  
  const API_URL = process.env.REACT_APP_API_URL;
  // This = 'http://localhost:8000/api'
  
  // Fetch data
  axios.get(`${API_URL}/events/`)
  
  // Create data
  axios.post(`${API_URL}/events/`, { title: 'New' })

Available endpoints on backend:

---

QUICK FIXES
===========

Backend not responding?
  docker-compose -f docker-compose-dev.yaml restart backend

Port already in use?
  For frontend: PORT=3001 npm start
  For backend: docker-compose down && docker-compose up -d

npm install problems?
  npm cache clean --force
  rm -rf node_modules package-lock.json
  npm install
